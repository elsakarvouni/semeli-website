A fully responsive, multi-lingual website designed and developed from scratch for **Semeli Italian Cuisine**, an authentic Italian restaurant. 

## 🚀 Project Overview

Initially conceptualized using a CMS (WordPress), I made the conscious architectural decision to pivot and develop this project entirely from scratch using native web technologies. This approach allowed for complete control over the UI design, superior loading speeds, and a fully optimized, bloat-free codebase.

The website serves as the digital storefront for the restaurant, providing essential information seamlessly across all devices.

## ✨ Key Features

*   **Multi-Lingual Support:** Fully localized interfaces and digital menus in Greek (EL), English (EN), and Italian (IT) to cater to both locals and tourists.
*   **Exceptional Performance:** Optimized image delivery and a semantic HTML structure resulting in near-perfect Google Lighthouse scores (100/100 in SEO, Accessibility, and Best Practices).
*   **Responsive Design:** A mobile-first approach ensuring the UI—including a custom hamburger navigation menu and image sliders—adapts flawlessly to any screen size.
*   **Interactive UI Components:** Features a custom CSS/JS image carousel and a language dropdown toggle without relying on heavy external libraries.
*   **Accessibility (a11y):** Screen-reader friendly with proper ARIA labels, semantic landmark tags (`<main>`, `<header>`, `<footer>`), and optimized color contrast.

## 🛠️ Tech Stack

*   **HTML5:** Semantic and accessible page structure.
*   **CSS3:** Custom styling, Flexbox/Grid layouts, and CSS animations.
*   **Vanilla JavaScript:** DOM manipulation for the mobile navigation menu, language toggles, and interactive elements.
*   **Hosting:** Deployed via GitHub Pages.
